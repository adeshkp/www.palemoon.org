Pale Moon Commander

Pale Moon Commander is a Mozilla Firefox compatible extension designed for the Pale Moon web browser. It is a configurator, meaning it will provide a user-friendly interface to advanced preferences that would otherwise require manual editing of parameters, which can be cumbersome and time-consuming to do.

How to access the add-on's settings

The add-on, when installed, adds a menu item "Advanced Options..." in two locations:

  • In the Application Menu (Pale Moon/Firefox button) under Options -> Advanced Options...
  • In the classic menu under Tools -> Advanced Options...

Clicking this menu item will open the preferences window that has been added by installing the add-on.

There be dragons here!

Using Pale Moon Commander to change advanced preferences is potentially dangerous and can leave your browser profile in a potentially unusable state. If you don't know what a setting is for, don't touch it. A number of the preferences made accessible through the add-on can prevent proper operation of the browser.

Because all the add-on does is change preferences, you can, at all times, reset the preferences by starting Pale Moon with the SHIFT key depressed, and selecting "Reset preferences to Pale Moon defaults". Of course this will also reset any other modified preferences you may have. Using the Reset feature in the add-on itself will have a similar result.

Use of this add-on is at all times at your own risk. I do not give warranty or guarantees, and it is provided "as-is" although I've done my best to make sure it works as-intended.

Download and install

The current version is 1.6.1 - this is an update adding settings for sessions, HSTS and tab focus control for javascript popups.
In addition, it will not automatically disable controls for features that are not applicable to your OS (e.g. Linux) or browser used (if installed in Firefox). The Swedish language has been updated and re-enabled in this point release.

You can go to the forum thread for this add-on to provide feedback.

You can go to the Mozilla Add-ons library where the add-on is listed (please check the version! If it's not the latest, go to "version information" at the bottom of the page, the click "view complete version history", and from there you can select the latest version that has not been reviewed yet by the editors):
