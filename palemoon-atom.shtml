Starting with Pale Moon 27.0, we no longer support Windows XP and other Windows versions from the same era (Server 2003, etc.) in any of our builds.

Further testing has also indicated that building specifically for Atom processors no longer provides a sufficient performance/maintenance cost tradeoff with the v27 code base.

All users previously on the Atom version will automatically be offered an upgrade to the main-line x86 build of Pale Moon, with the exception of Windows XP users who will not be offered an upgrade past the latest compatible version (determined by offered useragent string).
