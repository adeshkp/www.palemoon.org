Archived versions

This page provides access to archived versions of Pale Moon. These are older versions of the browser that are no longer supported, but listed here in case you want or need to use an older version. Please be aware that these versions may have (serious) bugs that have been fixed in later versions. We cannot reasonably provide support for too old versions.

Please only grab the version you personally need and do not hotlink.

Please understand that "anonymous" FTP access is not enabled; you must use the links with credentials as posted on this page. Trying to use the FTP server name by itself will not give you access to these downloads (this is required to deter scraper bots eating up our bandwidth for no real use).

Download (all old versions)

You can download archived versions of the browser from the Pale Moon archival FTP

ftp://palemoon:get@archive.palemoon.org/

If you need language packs for older versions, please use the following ftp link:
ftp://pmlocales:get@archive.palemoon.org/
