Advanced preferences: Update

This preferences tab provides advanced Pale Moon settings for updates.

Pale Moon updates

  • Automatically install updates: Firefox automatically downloads and installs found updates.
  • Check for updates, but let me choose whether to install them: Pale Moon lets you choose if you want to download and install updates. If you do not choose to do so, you can download them at a later time.
  • Never check for updates (not recommended): Pale Moon will not check for updates.
    Warning: If you select this setting, you will not receive critical security updates that may keep you safe online unless you frequently do a manual check for updates.

To review the history of updates that have been downloaded and installed, click Show Update History.

Note: You must be running Pale Moon as an administrator or as the user who originally installed the browser to install updates.

Automatically update
