Advanced preferences: Network

This preferences tab provides advanced Pale Moon settings for network connections and caching.

Connection

This allows you to configure how Pale Moon connects to the internet. Clicking the button will open an advanced connection settings dialog:

  • Configure Proxies to Access the Internet:
    One of five possible options:
    • No proxy: All connections are made directly to the server.
    • Auto-detect proxy settings: Pale Moon will attempt to determine the required connectivity settings for proxies automatically (either through a system proxy or direct connections).
    • Use system proxy settings: Use whatever proxy is set up in the operating system for Internet connectivity.
    • Manual proxy configuration: You enter proxy addresses and ports yourself. Please see the documentation or instructions given to you (usually by IT staff) what to enter in which location if you need this option.
    • Automatic proxy configuration URL: Enter the URL here of the proxy configuration file (.pac) you need to use. Please also see the relevant instructions or documentation given to you for this option.

Cached Web Content

By default, Pale Moon will store downloaded pages and content in its web cache for quick access without having to re-download data for pages on subsequent visits. This section informs you how much disk space is being used for this cached web content, and allows you to empty the cache and control its size.

Clear Now (button): This will empty the cache and remove all stored web content from disk.
Override automatic cache management: If you check this box, you can manually enter a maximum amount of disk space to use for cached web content. If unchecked, the browser will use default values up to a certain maximum based on your system specifications and available disk space.
Limit cache to: When manually setting the cache size, use no more than this amount of disk space for cache web content.

Offline Web Content and User Data

Web applications can store user data in your browser, and in some situations tell the browser to cache specific content so the web application can still be used even if you are off-line.

This section will tell you how much data is being stored on disk for this purpose, and give you control over which sites are allowed to store this data.

Clear Now (button): This will immediately clear stored user data and off-line web content. Please note that using this button, you may lose specific options/results stored in your browser by websites.
Pale Moon up to v27.1.1:
Tell me when a website asks to store data for off-line use: Controls how the browser responds to websites asking to store this kind fo data:
Checked: You will be prompted by the browser to allow storage of data (unless it's in the exceptions)
Unchecked: Storage of data is always denied
Pale Moon from v27.2 onwards:
When a website asks to store data for off-line use:
Allow by default: Storage of user data and offline web content is allowed without prompting (default), unless it's in the exceptions.
Always ask: You will be prompted by the browser to allow storage of data (unless it's in the exceptions)
Deny: Never allow storage of user data of off-line web content. Not recommended; this will break some web applications.
Exceptions (button): Opens a list with websites for which you have told the browser to block storage.
Section with user data storage entries: Shows a list of domains which have data stored in the browser. You can select and remove the data for individual entries.
