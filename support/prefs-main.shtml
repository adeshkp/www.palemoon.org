Startup, Home page and Downloads

This preferences page provides Pale Moon settings for browser startup, home page location, and several settings for downloads.

Browser startup

When Pale Moon starts, you can choose from the following startup behaviors:

  • Show my home page: This will make the initial page loaded when the browser starts the home page or pages set below. (default)
  • Show a blank page: This will make the initial page loaded when the browser starts an empty document.
  • Show my windows and tabs from last time: This will restore the browser windows and tabs that were open when you ended your previous browsing session (exited the browser).

Browser home page

In the Home Page field, you can enter any URL (or multiple, separated by a pipe | character) that the browser should treat as its home page. This home page will be used for the Home navigation button, and (if set) as startup page for browser start.
If you erase the contents of the field, the browser will use an (internal) home page with some basic functions and a web search bar.

The buttons allow you to quickly set the home page field:
  • Use current page: Sets the field to whatever URL is currently open in the browser window you opened the Options from.
  • Use Bookmark...: Opens a bookmark chooser, allowing you to pick one of the bookmarks you saved as a home page.
  • Restore to default: Resets the home page field to the default home page as shipped with the browser.

Downloads

This section allows you to choose where to save downloaded files. You can either select a default download location, making downloads start automatically and downloaded files to always be saved in the designated folder/directory, or select to have the browser ask where downloaded files should be saved every time.

(New in v27): On Windows, files downloaded from the internet can have specific zone information saved with them to mark the files as potentially unsafe. You can indicate here whether this zone information should be saved with downloaded files or not:

  • Never: The browser will never save zone information with the downloaded files (they are not marked as downloaded from the internet).
  • Always: The browser will always save this information with downloaded files.
  • Use system setting: The browser will use whatever the system-wide setting is for zone information to determine whether to save this information with downloaded files or not.
Note: Zone information can only be saved on NTFS drives.
