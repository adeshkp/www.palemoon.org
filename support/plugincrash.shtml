Plugin Crashed

This error means that a plugin (like Adobe Flash) has crashed. Simply reloading the page will restart the plugin, and your video (or other content) will be shown again.

What is a plugin?

A plugin is a piece of software that displays internet content that Pale Moon is not designed to display. This usually includes video, audio, online games and presentations that are made in patented formats. Plugins are created and distributed by the companies that make those patented formats. Some common plugins are Adobe Flash, Apple QuickTime, and Microsoft Silverlight.

What is a crash?

A crash happens when a piece of software abruptly stops working. Plugins sometimes crash for various reasons, and Pale Moon will display a notification that the plugin has crashed. On rare occasions, a plugin crash can also cause the entire browser to crash,  although this is usually prevented through the use of the plugin container.
