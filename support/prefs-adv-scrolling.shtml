Advanced preferences: Scrolling

This preferences tab provides advanced Pale Moon settings for smooth scrolling.

Use smooth scrolling: This master switch enables or disables smooth scrolling in the browser.

Smooth scrolling parameters

When smooth scrolling, you can decide when the browser scrolls smoothly, and if so, with which timings. By adjusting these timings, you can fine-tune the scrolling behavior and speed to your liking. You can indicate a set speed by having the two values be equal (time to scroll the distance per "tick") or an accelerated curve by having lower value (for the faster speed when e.g. scrolling the mouse wheel rapidly) and a higher value (for single "ticks") respectively in the input fields.
  • Smooth scroll with mouse wheel: This is the primary way of scrolling pages for most people. It is normal to use an accelerated curve here, so faster scrolling with the wheel results in faster scrolling of the page.
  • Smooth scroll with arrow keys: Smooth scrolling of the page when pressing the arrow keys to scroll the page content.
  • Smooth scroll with page up/down: Smooth scrolling when pressing PgUp or PgDn on the keyboard to scroll a full screen of page content.
  • Smooth scroll with scrollbars: Smooth scrolling when clicking the arrow keys on scrollbars
