Tabs

This preferences page provides Pale Moon settings for tabbed browsing.

The following selections are available:
  • Open new windows in a new tab instead: This option controls whether links from other applications or from web pages which request to open them in new windows are opened in a new window or a new tab in the most recent window.
  • Warn me when closing multiple tabs: When you close multiple tabs with a single action (e.g. "Close all other tabs" from the tab strip), Pale Moon will ask you to confirm this action when this option is checked, to prevent accidental closing of many pages at once.
  • Warn me when opening multiple tabs might slow down Pale Moon: When you open a large number of tabs at once, Pale Moon will ask you to confirm your choice. This prevents you from accidentally slowing down your system while the pages are loading. Uncheck this option to disable this warning.
  • Always show the tab bar: When this option is deselected, the tab bar will be hidden when you only have a single tab open.
  • Don't load tabs until selected: When checked, Pale Moon will only load the active tab when it starts or restores your previous session. This can make startup much faster if you are restoring many tabs. The other open tabs will be loaded as you click on them.
  • When I open a link in a new tab, switch to it immediately: When you middle-click on a Web link (or hold down Ctrl while clicking with the left mouse button), the page will be opened in a new tab. That page will not be displayed and will load in a background tab. Check this option to load and display the page in a new foreground tab instead.
  • Show tab previews in the Windows taskbar: When you move your pointer over the Pale Moon icon in the Windows 7 or later taskbar, Windows will show you a preview of the Pale Moon window. With this option check marked, Windows will show you a separate preview for each tab, instead.
  • Insert related tabs next to the current tab: When this option is checked, Pale Moon will insert tabs related to your active tab (e.g. a link from the current page) in a new tab inserted directly next to your active tab instead of at the end of the tab strip.
  • When performing a context search, switch to the new tab immediately: When you perform a context search (selecting a word/phrase, then right-clicking and choosing to search the web for that word/phrase), Pale Moon will open the search in a new tab. If you check this option, the search results tab will immediately be brought to the foreground.
  • Close the window when the last tab is closed: If you check this option, Pale Moon will close the browser window if you close the last tab in it. Otherwise, it will close the document, keep the window open, and show a new blank tab, instead.
  • When opening a new tab, show: Allows you to choose what newly-opened tabs display by default.
    • A blank page: Displays an empty document.
    • The Pale Moon start page: Displays the Pale Moon start portal.
    • The Quickdial page: Displays a page with thumbnails of the most used and/or pinned web pages.
