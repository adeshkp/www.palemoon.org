Download Pale Moon 25.7.2

New this release: This is a stability point release on top of the 25.7.1 security and usability update. Details about this update can be found in the Release notes.

These downloads are installers. At the bottom of the page, there are some links to zip packed versions of the browser meant for advanced use (not recommended for average use), that you can use to simply unpack to a location of your choice. If you are looking for a "portable" version of Pale Moon for removable media, click here.

Pale Moon is in the US English language
If you want to use Pale Moon in a different language, you can install a Pale Moon language pack. Instructions can be found here.

Minimum system Requirements:
  • Windows Vista/Windows 7/8/10/Server 2008 or later
  • A processor with SSE2 support
  • 256 MB of free RAM (512 MB or more recommended)
  • At least 150 MB of free (uncompressed) disk space
