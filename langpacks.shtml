Pale Moon language packs

If you prefer to have the Pale Moon user interface in your native language (not US-English), then these language packs can help you! They are based on the Mozilla Firefox language packs, with additional translations done for specific Pale Moon user interface components. Pale Moon will not be released with installers in individual languages, but installing these language packs on top of the US-English browser will pretty much have the same effect.

It should be noted that while these language packs convert most of the english in the user interface, many of them are not a complete translation as of now. Some Pale Moon specific options and texts are not localized (yet) in some languages and will show up as English if you use the packs marked as incomplete. If you want to help the Pale Moon project by translating these texts, please hop over to this forum board and help out!

Please note the installation instructions, just installing the language pack and letting Pale Moon restart is not enough! Also keep in mind that these language packs are a convenience and that the browser is and remains an English language product at its heart, so something like the Safe Mode dialog and default bookmarks folder names will be in English.

Installation instructions

A few simple steps is all that is needed to install these language packs. You have the choice of 2 different methods, either by installing the Zing extension or by using the instructions to perform a one-time preference change:

Extension method:
  1. Download the language pack .xpi from this page (below). Choose to immediately "install" in the Pale Moon browser (the default when left-clicking), skipping the need to save it first.
  2. Install the Zing! locale switcher from here:
    https://addons.palemoon.org/extensions/other/zing-locale-switcher/
  3. Click the Z! flag in your toolbar, and select the language you prefer from the drop-down.
  4. Let the browser restart when asked.
Preference method:
  1. Download the language pack .xpi from this page (below). You may also choose to immediately "install" in the Pale Moon browser (the default when left-clicking), skipping the need to save it first (go to step 3). You do not have to restart Pale Moon yet.
  2. If you downloaded the .xpi first, double-click the .xpi in explorer/other file manager. Confirm that you want to install the .xpi in your browser. This will add the language pack to Pale Moon. You do not have to restart Pale Moon yet.
  3. To actually switch to the new language, you also have to make a configuration change. Go to the advanced configuration editor (type about:config in the address bar and press enter).
  4. Find the setting general.useragent.locale which is set to "en-US" by default. Double-click it to change.
  5. Enter the language code for your locale, including region if applicable. E.g.: "fr" if you live in france, "ja" if you live in Japan, "es-MX" if you want Mexican Spanish. Use the same code as the file name of the language pack you downloaded.
  6. Close Pale Moon completely and restart it.

And that's it! You can now use Pale Moon in your native language.

Note that these language packs only change the interface language. They don't change the language used for the spellchecker.
To download a spellchecker dictionary of your choice, go to: https://addons.mozilla.org/en-US/firefox/language-tools/
 Or right-click any normal text input field, and in the pop-up menu select Languages -> Add Dictionaries...

These language packs are for the latest version of the browser (v26). They will not work on older versions. If you are using an older version of the browser, please update to the current version first. If you for whatever reason decide you need to use an older version of the browser, please go to the archived versions page and download the approporiate language pack for your browser version.

As of version 26, we've had to reduce the number of actively supported languages due to the big changes with this milestone and the lack of active translators for less common languages previously supported with a language pack.

Important note: These language packs are provided AS-IS. We, the authors of the Pale Moon browser, are not the authors of these language packs, and cannot provide support for the actual contents of them. You may want to check out the forum for some of the active translators for Pale Moon if you need to get in touch with them, but please understand that most of the existing translations are taken from past Mozilla localization and that this is a community effort.

Can't find your language?
We're sorry but these are the only languages that actually have a somewhat complete translation and have some support from native speakers. We cannot support any other languages without a dedicated community translator for a language.
