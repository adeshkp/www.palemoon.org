Pale Moon inactive language packs

The language packs listed below do not have a maintainer and are currently abandoned. This means that as time goes on, more and more parts of these language packs will be in English as Pale Moon continues to develop and the strings in the browser are changed/updated and added/removed to follow the browser's evolution.

Some Pale Moon specific options and texts are not localized and will show up as English if you use these packs. If you want to help the Pale Moon project by translating these texts, please hop over to this forum board and help out!

Please note the installation instructions on the main language packs page, just installing the language pack and letting Pale Moon restart is not enough! Also keep in mind that these language packs are a convenience and that the browser is and remains an English language product at its heart.

Download for v25.6-25.7

These language packs are for the latest version of the browser (v25.6). They will not work on older versions. If you are using an older version of the browser, please go to the archived versions page and download the approporiate language pack for your browser version.
