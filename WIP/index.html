Pale Moon Works in Progress

Important: these builds for testing purposes only and are at all times used at your own risk. They may be unstable and may crash, do unexpected things, erase your files or set your hair on fire. Do NOT expect release stability from these!

That being said, if a file is here for more than a few days, it's likely stable enough for daily use.

These files are mainly here for beta testing before release, or to test out new and/or changed functionality in the browser, as well as tryouts of other Mozilla packages or for other Operating Systems, or work-in-progress public milestone betas.

If you wish to give feedback to these works-in-progress, please use the forum (see below for links to dedicated threads).

  • Pale Moon 26.0.0, public beta 4

    This is the fourth and final public beta of the new milestone release. Considering the large amount of changes involved, there will be some issues with websites, although at the moment they seem to be mostly centered around useragent sniffing.

    This beta fixes applicable security vulnerabilities, adds promises, adds an XSS filter, and much more! See the Goanna release notes for details.

    To provide feedback on this browser, please use the dedicated support topic on the forum. Unfortunately we cannot accept any feedback through other channels because it would result in utter chaos for us. So please, do not use e-mail, do not use social media, do not use IRC to provide feedback to this beta, and help keep things organized for us! :)

    Known issues on sites:
    • Some extensions that check the platform version may be dysfunctional, because it's currently exposed as "2.0" to reflect the 2.0 version of the Goanna platform. Extensions will likely need to be updated for proper support, since there are many ways to check for platform version and we can't reasonably create "fake" values for all of them.
    • If you find other major issues, please let us know so we can add them here.
