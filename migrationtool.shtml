Pale Moon profile migration tool


Note: with the increasing divergence from Firefox, this migration tool has become unusable in its current state. If you wish to import individual items from Firefox to Pale Moon like bookmarks, you are recommended at this time to use available import/export features in either browser (e.g. export to html for bookmarks) or external tools, and otherwise re-build your profile from a fresh installation of Pale Moon.
We are looking into a more functional replacement of this tool, but no timeline is set for completion of it at this time.
